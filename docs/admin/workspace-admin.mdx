# Workspace Administration

Workspace admins configure security, structure, and operational standards.

## Membership and access

- Invite members and assign roles
- Manage external collaborators
- Set project-level scoped access for selected members

## Workspace settings

- Name, slug, description, website
- Branding (logo and identity)
- Collaboration policy settings

## Project governance

- Create and archive projects
- Configure project settings and module availability
- Review team membership and activity logs

## Billing and plans

Billing plan affects feature access:

- Pro features for advanced construction workflows
- Enterprise/O&M features for operations modules

## Localization

PlanSync supports multiple languages, including RTL for Arabic.

Supported locales include English, French, Danish, Arabic, Spanish, and Simplified Chinese.

## Notification and compliance posture

- Use audit logs for change history
- Configure push notifications where applicable
- Review access and role assignments regularly
