# Client and Proposal Portals

PlanSync includes external portal experiences for collaboration without full internal access.

## Client portal

The client portal can expose selected project modules, commonly:

- Drawings/files
- Issues
- Reports
- Proposals

Visibility is controlled by project/workspace settings.

## Proposal portal

Proposal share links are tokenized and support:

- Viewing proposal details
- Acceptance/decline/change request actions
- Message threads between client and team

## Occupant portal

Occupant token links allow issue submission for O&M/tenant workflows.

## Invite and deep link flows

PlanSync includes join/open routes for:

- Invite acceptance
- Email invite acceptance
- Open-from-email routing

## Best practices

- Share links through trusted channels only
- Revoke/rotate tokens when exposure is suspected
- Keep module visibility minimal for each client audience
