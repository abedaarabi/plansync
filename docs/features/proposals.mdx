# Proposals

PlanSync proposals connect estimate data with client communication.

## Proposal lifecycle

Common lifecycle states:

- Draft
- Sent
- Viewed
- Accepted / declined
- Change requested

## Core capabilities

- Build proposal line items (manual or from takeoff sources)
- Preview and generate proposal documents
- Send/resend to clients
- Duplicate for iterative workflows
- Track revisions, comments, and portal messages

## Proposal templates

Workspace-level templates standardize:

- Cover copy and structure
- Default values and assumptions
- Branding presentation

## Public proposal portal

Client actions are supported through public tokenized routes:

- View proposal
- Accept
- Decline
- Request changes
- Exchange messages

Keep proposal URLs confidential because token possession grants access.
