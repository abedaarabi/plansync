# Construction Workflows

PlanSync supports end-to-end construction coordination from drawing review through closeout.

## Issues

Use issues to track field conditions, clashes, defects, and follow-up actions.

- Assign owners and due dates
- Link to drawing positions and markups
- Track statuses and priorities

## RFIs

Manage Requests for Information with response tracking.

- Create RFIs from drawing context or issue links
- Assign responders and set due dates
- Capture official responses and attachments

## Punch lists

Manage completion and quality items near handover.

- Create and assign punch items
- Track status and completion evidence
- Export punch data when needed

## Field reports

Capture recurring site updates and distribute by email.

- Create project field reports
- Update report details
- Send report summaries to recipients

## Schedule

Store and update project schedule tasks in PlanSync to align production data with field execution.

## Audit logs

Every critical mutation can be traced in project/workspace logs to support accountability and compliance.
